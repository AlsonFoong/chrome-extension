# chrome-extension
A customized Chrome extension built for personal use.  

**Core Stack**: TypeScript/JavaScript/HTML/CSS  
**Why this stack**: JavaScript was my first language as a self-taught programmer, so this pet project served as a playground for my newfound skillset at the time, serving as a general entry point into the world of programming. TypeScript is a superset of JavaScript that adds type checking, and was integrated both to improve the developer experience and to grow my skillset.  
  
**Features**:
- Intercepts API calls made by the browser to track user activity and send alerts
- Manipulates DOM elements to enhance website experience
- Manages state and user data using Chrome's storage APIs
